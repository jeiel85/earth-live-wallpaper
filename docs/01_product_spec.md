# 01 Product Specification

## Product Name

EarthLive Wallpaper

## One-line Description

Windows 11용 위치 기반 지구 라이브 월페이퍼로, 현재 위치와 시간에 따라 지구의 낮/밤 경계선, 일출/일몰, 위성/날씨 레이어를 시각화한다.

## Target User

- Windows 11 데스크톱 사용자
- 지구/우주/날씨 시각화에 관심 있는 사용자
- 미니멀하면서도 정보성이 있는 라이브 월페이퍼를 원하는 사용자
- 한국/동아시아 날씨와 태풍 흐름을 시각적으로 보고 싶은 사용자

## Core Value

단순히 예쁜 움직이는 배경이 아니라, **내 위치와 실제 시간에 맞춰 변화하는 살아있는 지구 배경**을 제공한다.

## MVP Value Proposition

첫 버전은 다음 한 문장으로 설명 가능해야 한다.

> "현재 시간의 지구 낮/밤 경계선과 내 위치의 일출/일몰을 보여주는 Windows 11 라이브 월페이퍼."

## MVP User Stories

### Story 1: 첫 실행

사용자는 월페이퍼를 실행하면 3D 지구가 표시되고, 현재 시간 기준으로 낮과 밤의 영역이 보인다.

### Story 2: 위치 설정

사용자는 위치 권한을 허용하거나, 권한을 거부한 경우 직접 도시명/위도/경도를 입력할 수 있다.

### Story 3: 내 위치 보기

사용자는 지구 위에서 자신의 위치 핀을 볼 수 있다.

### Story 4: 일출/일몰 보기

사용자는 화면 한쪽 정보 패널에서 현재 위치의 일출 시간, 일몰 시간, 현재 태양 고도 정보를 볼 수 있다.

### Story 5: Lively Wallpaper 등록

사용자는 빌드된 웹 월페이퍼를 Lively Wallpaper에 등록해서 Windows 바탕화면으로 사용할 수 있다.

## Non-goals for MVP

- 실시간 태풍 경보
- 공식 재난 알림
- 구글어스 포토리얼 3D
- 건물 그림자
- 자체 Windows 월페이퍼 엔진
- 서버 기반 계정/동기화
- 광고/분석/추적

## UX Tone

- 조용함
- 과학적
- 세련됨
- 정보는 작고 절제되게
- 지구와 빛의 움직임이 주인공이어야 함

## Default Visual Concept

- 배경: 깊은 우주색
- 지구: 천천히 회전
- 낮 영역: 자연광
- 밤 영역: 어둡지만 도시광 또는 은은한 야간 느낌
- 현재 위치: 작고 선명한 핀
- 정보 패널: 반투명 glassmorphism 스타일
