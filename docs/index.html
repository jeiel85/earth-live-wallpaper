EarthLive Wallpaper

A stunning real-time 3D Earth live wallpaper for Windows 11. Location-aware sunlight boundary, responsive, and privacy-first.

Download Release View on GitHub

Key Features

🌍

3D Earth Globe

Powered by CesiumJS, rendering a highly detailed 3D globe with silky smooth auto-rotation and orbit control.

☀️

Sunlight Terminator

Visualizes the real-time day and night boundary across the Earth, adapting instantly based on your local coordinates.

🔒

Privacy-First

Your location data is stored entirely locally on your machine. Absolutely zero data is sent to external servers.

⚙️

Lively Integration

Built directly for the Windows 11 Lively Wallpaper engine. Smooth operation with ultra-low hardware footprints.
